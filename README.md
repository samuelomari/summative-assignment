# Wordly Dictionary App

A clean, minimalist dictionary application built for a summative assignment. This project uses the Free Dictionary API to provide users with word definitions, parts of speech, pronunciations, and audio samples.

## Features
- **Instant Search:** Fetches real-time data from the Free Dictionary API.
- **Audio Support:** Listen to the pronunciation of words (when available).
- **Phonetics & Synonyms:** Displays phonetic spelling and up to 4 synonyms.
- **User Experience:** Automatically clears the input field and focuses for the next search.
- **Error Handling:** Friendly alerts if a word cannot be found.

## Tech Stack
- **HTML5:** Semantic structure for the dictionary interface.
- **CSS3:** Custom styling for a modern, responsive "word-card" layout.
- **JavaScript (ES6+):** - `Fetch API` for asynchronous data handling.
    - `Async/Await` for clean API requests.
    - `DOM Manipulation` to dynamically update search results.

## Project Structure
- `index.html`: The main entry point.
- `style.css`: All layout and design rules.
- `main.js`: Logic for fetching data and displaying results.

## How to Use
1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
3. Enter a word in the search box and click **Search**.
4. Explore the definitions and listen to the audio playback!
